# Structural Safety Check (Jordan)

A professional preliminary building safety assessment tool (Jordan) built with React + Vite.

## Features
- Professional UI design
- Arabic + English support
- Questions with scientific basis (ACI 318, ASCE 7)
- Results with risk classification (Green/Yellow/Red)
- PDF report generation
- Case studies section
- Privacy & Disclaimer pages
- Jordan illustrative seismic map

## Setup
```bash
npm install
npm run dev
